# Demo Maker

Generate narrated MP4 product demos from your codebase — for GitHub, Twitter, Product Hunt, Instagram, TikTok, and more.

## Install

### Cursor
```bash
git clone https://github.com/julieclarkson/demo-maker.git .demo-maker
```
Then copy `cursor/` contents into your project.

### Claude Desktop (Cowork)
```bash
git clone https://github.com/julieclarkson/demo-maker.git .demo-maker
```
Then copy `claude/` contents into your project.

### Bundle (all three plugins)
- **Cursor**: [launchpad-cursor](https://github.com/julieclarkson/launchpad-cursor)
- **Claude**: [launchpad-claude](https://github.com/julieclarkson/launchpad-claude)

## Structure

```
cursor/   — Cursor plugin (rules, prompts, scripts, Remotion project)
claude/   — Claude plugin (skills, commands, prompts, scripts, Remotion project)
```
